# 教科別 学習クイズ

国語と社会のクイズ形式の学習サイトです。

## 機能

- クイズ形式で問題にチャレンジ
- 学習履歴の記録・確認
- スマホ対応のレスポンシブデザイン
- ローカルストレージで履歴を保存

## 使い方

1. 依存パッケージをインストール
```bash
npm install
```

2. 開発サーバーを起動
```bash
npm run dev
```

3. ブラウザで `http://localhost:5173` を開く

## ビルド

```bash
npm run build
```

## 技術スタック

- React
- TypeScript
- Vite
